# SQL Portfolio Risk Diagnostics

SQL-based financial risk engineering framework using MySQL.

---

## Overview

This project builds a client-level portfolio risk diagnostic workflow using SQL.

It evaluates:

- Multi-horizon returns (12M / 18M / 24M)
- Volatility (sigma)
- Average daily returns
- Risk-adjusted return ratios
- Asset allocation structure
- Market-regime aware investment recommendations

---

## Architecture

MySQL → SQL Modeling → Risk Metrics → Visualization (Tableau) → Strategic Interpretation

---

## Repository Structure

- `sql/` – SQL scripts (view creation + metrics)
- `results/` – Visual outputs (charts)
- `report/` – Written analysis (PDF version)
